# 15.4 연속 처치·다중 임계값 확장

이 절은 **15.4 연속 처치·다중 임계값 확장**의 핵심 개념과 식별 가정, 해석상의 주의점을 연결해 정리한다.

## 15.4 연속 처치·다중 임계값 확장
현실의 정책은 하나의 cutoff와 하나의 이진 처치로 끝나지 않는 경우가 많다. 장학금이 점수 구간별로 금액이 달라지거나, 복지급여가 자격 경계마다 단계적으로 바뀌거나, 세금·보조금이 절단점에서 수준(level)이 아니라 기울기(slope)를 바꾸는 경우가 그렇다. 이 절에서는 그런 확장형 RDD를 본다.

## 왜 확장형이 필요한가

기본 sharp/fuzzy RDD는

\[
R \text{ around } c
\]

에서 하나의 불연속만 본다. 그러나 정책 현장에서는 다음 구조가 흔하다.

- **다중 cutoff**: 지역별, 연도별, 제도별로 cutoff가 여러 개
- **연속 또는 다단계 처치**: cutoff를 넘는 정도에 따라 혜택 크기가 달라짐
- **regression kink design (RKD)**: 수준 점프가 아니라 기울기 변화가 핵심

따라서 15.4는 “RDD는 꼭 하나의 문턱만 보는가?”라는 질문에 대한 답이라고 볼 수 있다.

## 다중 cutoff RDD

어떤 정책은 동일한 제도를 여러 지역이나 집단에서 서로 다른 cutoff로 적용한다. 예를 들어 지역별 소득 기준이 다르거나, 학교별 선발 기준이 다르거나, 연도별 점수 기준이 조정되는 경우가 그렇다.

이때 각 cutoff \(c_j\)마다 국소 효과

\[
\tau(c_j)
\]

를 따로 식별할 수 있다. 중요한 점은 이를 함부로 하나의 숫자로 묶지 않는 것이다. 각 cutoff는 서로 다른 국소 집단을 대표하므로, 효과가 다를 가능성이 크다.

정책평가에서 좋은 실무는 다음과 같다.

- cutoff별 결과를 먼저 분리 보고
- 제도적으로 cutoff들이 비교 가능한지 설명
- 통합 추정치를 보고한다면 어떤 가중으로 평균했는지 명시

예를 들어 여러 소득구간마다 복지급여 기준이 다른 경우, 각 구간은 서로 다른 생활수준과 취약성을 가진 집단일 수 있다. 이들을 단순 평균한 값은 해석이 모호할 수 있다.

## 연속 처치 RDD

어떤 정책은 cutoff를 넘으면 처치 여부만 바뀌는 것이 아니라, 처치 강도 자체가 달라진다. 예를 들어 성적이 높을수록 장학금 액수가 늘거나, 자격점수가 낮을수록 보조금 수준이 커지는 식이다.

이 경우 결과 \(Y\)에 대한 식별은 단순 이진 처치 효과라기보다, cutoff 근방에서 처치 강도가 바뀌는 국소적 한계효과에 가깝다. 처치강도 \(D\)가 연속적이면, cutoff에서의 평균 처치강도 점프를 이용해 결과 점프와 함께 해석할 수 있다.

만약 cutoff에서 처치강도가 불연속적으로 점프한다면 fuzzy RDD의 비율 논리를 연속 처치에도 적용할 수 있다.

\[
\tau_{cont}
=
\frac{
\lim_{r\downarrow c}\mathbb{E}[Y\mid R=r]
-
\lim_{r\uparrow c}\mathbb{E}[Y\mid R=r]
}{
\lim_{r\downarrow c}\mathbb{E}[D\mid R=r]
-
\lim_{r\uparrow c}\mathbb{E}[D\mid R=r]
}
\]

즉 cutoff가 유발한 처치강도 변화 1단위당 결과 변화로 읽을 수 있다.

## Regression kink design (RKD)

어떤 정책은 cutoff에서 수준이 점프하지 않고, 기울기만 바뀐다. 예를 들어 소득이 증가할수록 급여가 점점 줄어드는데, 특정 구간에서 감액률이 갑자기 바뀌는 경우가 그렇다. 이때는 수준의 불연속이 아니라 **도함수의 불연속**을 이용한다.

\[
\tau_{RKD}
=
\frac{
\lim_{r\downarrow c}\frac{d}{dr}\mathbb{E}[Y\mid R=r]
-
\lim_{r\uparrow c}\frac{d}{dr}\mathbb{E}[Y\mid R=r]
}{
\lim_{r\downarrow c}\frac{d}{dr}\mathbb{E}[D\mid R=r]
-
\lim_{r\uparrow c}\frac{d}{dr}\mathbb{E}[D\mid R=r]
}
\]

정책평가에서 세금제도, 보조금 tapering rule, 급여 감액 규칙 분석이 여기에 자주 해당한다. RKD는 구현과 해석이 더 어렵지만, 수준점프가 없는 제도에서도 유용한 국소 식별을 제공한다.

## 여러 cutoff와 외삽의 문제

다중 cutoff가 있다고 해서 그것을 자동으로 결합해 “더 일반적인 효과”를 얻는 것은 아니다. 각 cutoff의 국소 효과는 서로 다른 국소 집단에 대한 것이다. 따라서 cutoff별 효과 차이는 단순한 소음이 아니라 실제 이질성일 수 있다.

정책평가에서 이는 매우 중요하다. 예를 들어 성적 80점 cutoff와 95점 cutoff 주변 학생은 전혀 다른 집단일 수 있고, 복지 자격점수 낮은 경계와 높은 경계 근방 가구도 매우 다를 수 있다. 그래서 통합 추정치를 제시하더라도, 그것이 어떤 집단들을 어떤 가중으로 평균한 것인지 설명해야 한다.

## 실무 쟁점: 정규화와 pooling

여러 cutoff를 함께 분석할 때는 running variable을 각 cutoff 기준으로 재정렬하여

\[
\tilde R_i = R_i - c_j
\]

처럼 centered running variable을 만들고 pooled analysis를 하기도 한다. 하지만 이때도 다음을 조심해야 한다.

- cutoff별 기초 분포가 비슷한가
- 결과 함수의 형태가 cutoff마다 크게 다르지 않은가
- pooled bandwidth가 개별 cutoff 분석을 왜곡하지 않는가

즉 pooling은 편리하지만, 이질성이 큰 제도를 억지로 하나의 로컬 설계처럼 보이게 만들 위험도 있다.

## 정책평가 사례

확장형 RDD는 여러 정책에서 자연스럽게 나타난다.

- **장학금 금액의 단계적 증가**: 점수 구간에 따라 장학금 액수가 달라지는 경우
- **세금/보조금 기울기 변화**: 소득 구간별 세율·감액률이 바뀌는 경우
- **연도별 상이한 cutoff**: 같은 제도가 해마다 기준을 조정하는 경우
- **지역별 자격 기준**: 지방자치단체마다 별도 cutoff를 적용하는 경우

이런 사례에서는 “정책이 있는가/없는가”보다 “정책 강도가 어떻게 바뀌는가” 또는 “경계가 몇 개나 있는가”가 더 중요한 질문이 된다.

## 정리

RDD의 확장형은 현실 정책의 복잡성을 더 잘 반영한다. 다중 cutoff는 여러 국소 효과를, 연속 처치와 kink 설계는 수준점프 대신 강도나 기울기 변화를 이용한다. 그러나 설계가 확장될수록 식별은 더 풍부해지는 대신, 해석과 pooling의 주의점도 커진다. 따라서 각 cutoff와 각 설계가 정확히 누구의 어떤 국소 효과를 식별하는지 분명히 해야 한다.
