# 19.1 예측과 인과의 목표 차이ML은 \(P(Y\mid X)\) 예측에 강하지만, 인과 추론의 목표는 \(P(Y\mid do(D))\) 또는 \(E[Y(1)-Y(0)]\)다.
      따라서 높은 예측 성능과 인과 식별은 별개의 문제다. 이 장은 두 세계를 연결하는 실무적 문법을 정리한다.

## 19.1 예측과 인과의 목표 차이      예측은 손실함수 최소화가 목표이고, 인과는 반사실 효과의 식별·추정이 목표다.
      같은 모델이라도 목적함수와 가정이 다르면 해석이 달라진다.

    \[
      \text{Prediction target: } f^*(x)=E[Y\mid X=x], \qquad
      \text{Causal target: } \tau(x)=E[Y(1)-Y(0)\mid X=x].
    \]

:::
